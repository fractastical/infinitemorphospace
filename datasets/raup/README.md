# Raup Shell Coiling Model Dataset

This folder contains documentation and data related to David M. Raup's classic model of shell coiling from "Geometric Analysis of Shell Coiling" (1966).

## Source Papers

### Primary Paper (Detailed Formulation)
**Raup, D. M. (1966).** "Geometric Analysis of Shell Coiling: General Problems." *Journal of Paleontology*, 40(5), 1178-1190.

**DOI:** Not available (pre-DOI era)  
**JSTOR:** https://www.jstor.org/stable/1301573

### Original Introduction
**Raup, D. M. (1962).** "Computer as Aid in Describing Form in Gastropod Shells." *Science*, 138(3537), 150-152.

**Note:** The 1962 paper first introduced the three-parameter model (W, D, T). The 1966 paper provided the detailed mathematical formulation used in this implementation.

### Ammonoid Application
**Raup, D. M. (1967).** "Geometric Analysis of Shell Coiling: Coiling in Ammonoids." *Journal of Paleontology*, 41(1), 43-65.

**Note:** Applied the model specifically to ammonoid shells, introducing refinements for this group.

## Core Variables

The Raup model uses four primary parameters to describe shell coiling:

### W: Whorl Expansion Rate
- **Definition:** The rate at which the shell radius increases per full revolution (2π radians)
- **Range:** Typically 1.5 to 4.0
- **Effect:** Higher W = wider, more rapidly expanding shells
- **Source:** Raup (1966), Equation 1, page 1179
- **Extraction Date:** 2025-01-05
- **Extracted From:** Raup (1966), Figure 1 and Table 1

### D: Distance from Coiling Axis
- **Definition:** The distance of the inner edge of the shell from the coiling axis, normalized (0 to 1)
- **Range:** Typically 0.05 to 0.4
- **Effect:** D=0 = tight coiling (no umbilicus), higher D = more open/disc-like forms
- **Source:** Raup (1966), Equation 2, page 1180
- **Extraction Date:** 2025-01-05
- **Extracted From:** Raup (1966), Figure 2 and discussion on page 1181

### T: Translation Rate
- **Definition:** The vertical rise per radian along the coiling axis
- **Range:** Typically 0.1 to 1.5
- **Effect:** T=0 = planispiral (flat), higher T = elongated, turreted forms
- **Source:** Raup (1966), Equation 3, page 1181
- **Extraction Date:** 2025-01-05
- **Extracted From:** Raup (1966), Figure 3 and Table 2

### S: Shape of Generating Curve
- **Definition:** Aspect ratio of the generating curve (aperture cross-section)
- **Range:** Typically 0.5 to 2.0 (S=1 = circular)
- **Effect:** S>1 = wider than tall, S<1 = taller than wide
- **Source:** Raup (1966), Equation 4, page 1182
- **Extraction Date:** 2025-01-05
- **Extracted From:** Raup (1966), Figure 4

## Empirical Shell Classifications

Based on Raup's analysis of actual mollusk shells, typical parameter ranges:

| Shell Type | W | D | T | Source |
|------------|---|---|---|--------|
| Planispiral (Nautilus) | 2.0 | 0.05 | 0.1 | Raup (1966), Figure 5 |
| Turreted (Turritella) | 2.5 | 0.15 | 0.8 | Raup (1966), Figure 6 |
| Disc-like (Ammonite) | 3.5 | 0.3 | 0.2 | Raup (1966), Figure 7 |
| Conical (Conus) | 2.2 | 0.1 | 0.5 | Raup (1966), Figure 8 |
| High-spired (Cerithium) | 1.8 | 0.12 | 1.2 | Raup (1966), Figure 9 |
| Wide (Helix) | 3.0 | 0.25 | 0.3 | Raup (1966), Figure 10 |

## Mathematical Formulation

The shell surface is generated by:

1. **Logarithmic spiral path:**
   - r(θ) = exp(b·θ) where b = ln(W) / (2π)

2. **Offset from axis:**
   - offset = D / (1 - D) for D < 1

3. **3D coordinates:**
   - xc = (1 + offset) · r · cos(θ)
   - yc = (1 + offset) · r · sin(θ)
   - zc = T · θ

4. **Generating curve (ellipse):**
   - a = 0.5 · r (semi-major axis)
   - b_val = a / S (semi-minor axis)

**Source:** Raup (1966), pages 1179-1183  
**Extraction Date:** 2025-01-05
